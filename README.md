# Btechnics IOT

Officiële distributie van de Btechnics IOT desktop applicatie voor macOS en Windows.

## Over

Btechnics IOT is een desktop applicatie die fungeert als slimme wrapper rond het Btechnics IOT systeem. De app levert native push notificaties, een rules engine voor melding routing, do not disturb modus, snooze functionaliteit en een geschiedenis van events.

## Installatie

De installers voor macOS (signed en notarized) en Windows zijn beschikbaar onder de [Releases](https://github.com/bogaertm/btechnics-iot-app/releases) pagina.

Kies de juiste download voor jouw platform:

**macOS**: Btechnics IOT [versie] universal mac.zip (werkt zowel op Intel als Apple Silicon)

**Windows**: Btechnics IOT Setup [versie].exe

## Auto update

De applicatie controleert automatisch op nieuwe versies via deze repository en biedt updates aan in de app zelf. Een handmatige herinstallatie is dus enkel nodig voor de eerste installatie.
